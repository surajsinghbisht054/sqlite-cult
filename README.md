# SQLITE-CULT is a web-based SQLite database browser written in python.

Project dependencies:
- Django
- Pygments

### Features

- Can create and update sqlite3 databases.
- Add or drop:
	- Tables
	- Columns
	- Indexes   
- Export data as JSON or CSV.
- Import JSON or CSV files.
- Browse table data.
- Insert, Update or Delete rows.

### Benefit from Django
- Authentication & Authorization
- Permission
- Class-Based Views
- Well Structured Code 
- Extensively Used Forms, Generic Views Etc
- Pure CSS/JS Code (No library used)
